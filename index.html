jQuery Hide / Show

Inserts an accessible buttons/links to hide and show sections of content.

Example A

Example of the hide/show plugin with the default options.

Text content to be hidden text content to be hidden text content to be hidden text content to be hidden

Initialisation


$(".exampleA").hideShow();

Example B

Example of the hide/show plugin with customised options. The content should be hidden by default.

Initialisation


$(".exampleB").hideShow({
    speed: '600',
    showText: 'Expand Search',
    hideText: 'Collapse Search',
    state: 'hidden',
    triggerElement: true
});

Example C

Example showing an existing element on the page converted to a trigger element for the hide/show content.

Can't find your doctor?

Please call 01234 567 890.

Initialisation


$(".exampleC").hideShow({
    speed: '600',
    state: 'hidden',
    triggerElement: true,
    triggerType: 'button'
});
